Videogames

All the repositories belonging in this section comply with the following nomenclature scheme:
  • The first part of the repository name denotes a game unique identifier, that consists of four alphanumeric characters.

    In case the game name is unique across all consoles/platforms, it's computed by taking the first four letters or numbers of the first four words that compose its title. In case the game title is composed of less than four words, it's padded at the end with more characters from the last word.

    In case the game name is not unique across all consoles/platforms or it's unique and the title is composed overall by less than four alphanumeric characters (but more than one) in total, the precedent rule applies, but with a twist. Only the first two alphanumeric characters obtained with the precedent method are taken in consideration as will be padded at the end by a console/platform identifier (the "short" ones in the table below).

    In case the game name consists by only a single alphanumeric character, is padded at the end by a longer console/platform identifier (the "long" ones in the table below).

    Console/Platform Identifier Notes
    Short Long
    Nintendo 64 64 n64
    Nintendo GameCube gc gcn GCN is Nintendo's official acronym for it
    Sega Genesis/Mega Drive ge gen The US name is chosen, for being most widely known

    Such identifiers are chosen to be the best compromise between easy recognizability and uniqueness. Long identifiers are made by basically adding another alphanumeric character to short ones.

  • The second part details, with only one word, what aspect of the videogame the repository is about. In cases where only one word might generate confusion, another one is used alongside it.

  • The third and last part describes, with only one word, specifically what is contained in the repository does regarding the aspect of the videogame detailed before.
